🔃 Token sẽ tự động được exchange: User Token → Long-lived Token → Page Access Token
💡 Hướng dẫn lấy token:
  1. Truy cập Facebook Graph API Explorer
  2. Chọn ứng dụng của bạn
  3. Chọn quyền: pages_messaging
  4. Generate Access Token (User Token)
  5. Paste token vào đây, hệ thống sẽ tự động lấy Page Token

🔗 Webhook Configuration

⚙️ Cấu hình webhook trên Facebook:
  1. Vào Facebook Developer Console
  2. Messenger → Settings → Webhooks
  3. Callback URL: /webhook (đường dẫn tương đối)
  4. Verify Token: {{ verify_token }}
  5. Chọn các events: messages, messaging_postbacks
